ACS-85-0510 16 bit Shift Register  Github

Short Shift register

These are used in lunettas outputed to r2r ladders sometimes. One bit delay perhaps. This Sketch will not give you any more functionality that a CD4014, but its a good start for you to add on for other similar sketches.

This version is just a simple register using a long and outputing the 0, 4 and 8th bits. Internally, its a unsigned int so thats 16 bits.
