ACS-85-0530 Drunk Walk  Github

simulation of the drunk walk as an audio output. Start at a set frequency and randomly take a step up or down.

Uses the LFSR to determine a random number that we compare against. If we are above or below that number, increment or decrement the frequency respectively. You do get a very wondering effect. Although the responsiveness of the parameters leave a little to be desired. You can tell that you have the skew set or stagger on...but its not too much different. Its not an gripping knob turner..but certainly ambient changes would work well here.

Standard walk. Only vary the speed of the register.
Walk with stagger turned on. This varies the increment size.
Adds the skew. This alters the probability so that it generally goes up or down.
