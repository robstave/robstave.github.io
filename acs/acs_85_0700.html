ACS-85-0700 Analog to Digital Decoder  Github

This sketch takes an analog input and converts it to a digital output on 3 pins. Either as a single pin or 3 bit binary mode.

Binary mode (PB4 = HIGH) It is really just a 3 bit AD converter. The value on the analog pin is converted to 3 bits. {000, 001, 010, ..., 111}

Single select mode (PB4 = Low) The value converts to one of three values. {001, 010, 100} Can be used to select things as well. This can be used to convert a pot to control channels, gates or whatever. An example below uses the sketch and nands to make a signal selector.
